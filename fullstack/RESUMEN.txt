      ├─ Selección de rol (Ventas, Planeación, Compras, Logística)
      ├─ Asignación opcional de empresa

      ├─ Código de 6 dígitos enviado por email
      ├─ Validez de 15 minutos
      ├─ Opción de reenviar código
      ├─ Diseño HTML profesional
      └─ Verificación requerida antes de login

  3️⃣  RECUPERACIÓN DE CONTRASEÑA
      ├─ Contraseña temporal de 10 caracteres
      ├─ Envío por correo electrónico
      ├─ Validez de 24 horas
      └─ Opción de cambiar después de login

  4️⃣  CAMBIO DE CONTRASEÑA
      ├─ Ruta protegida para usuarios autenticados
      ├─ Validación de contraseña actual
      ├─ Confirmación de nueva contraseña
      └─ Mínimo 6 caracteres

┌──────────────────────────────────────────────────────────────────────────────┐
│ 🌐 NUEVAS RUTAS                                                             │
└──────────────────────────────────────────────────────────────────────────────┘

  Ruta                          Métodos      Descripción
  ─────────────────────────────────────────────────────────────────────────────
  /auth/login                   GET, POST    Login (actualizado)
  /auth/register                GET, POST    Registro de usuario
  /auth/verify-email            GET, POST    Verificación de correo
  /auth/resend-verification     POST         Reenviar código
  /auth/forgot-password         GET, POST    Recuperar contraseña
  /auth/change-password         GET, POST    Cambiar contraseña
  /auth/logout                  GET          Cerrar sesión

     └─ python app.py

└──────────────────────────────────────────────────────────────────────────────┘

  📝 REGISTRO NUEVO USUARIO
     1. Click "Crear Cuenta" en login
     2. Completar formulario con correo institucional
     3. Recibir código de 6 dígitos
     4. Verificar código
     5. Iniciar sesión ✅

  🔐 RECUPERAR CONTRASEÑA
     1. Click "Recuperar" en login
     2. Ingresar correo electrónico
     3. Recibir contraseña temporal
     4. Iniciar sesión con contraseña temporal
     5. Cambiar contraseña 🔒

  👨‍🎓 USUARIO PRUEBA PILOTO
     1. Ingresar estudiante_X_Y
     2. Login directo (sin contraseña)
     3. Acceso inmediato ⚡
